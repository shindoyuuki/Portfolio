学習したスキル

1.

HTML

HTMLの最終課題では、スクールHPの模写を実装致しました。いきなりの模写で大変でしたが、基本を抑える事ができました。

2.

CSS

メディアクエリやハンバーガーメニュの実装はとても難しかったです。ネットで調べながら進める事ができました。cssでHPの顔が決まるんだなと実感致しました。

3.

JavaScript
